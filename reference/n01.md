# Multivariate standard normal data

A synthetic data set containing 1000 observations on 10 variables
generated from independent standard normal distributions.

## Usage

``` r
n01
```

## Format

A data frame with 1000 rows and 10 columns.

## Value

Data frame

1.4, <https://OTexts.com/weird/>.
